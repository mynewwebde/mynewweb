# MNW Logo & Favicon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add the MNW monogram logo (classic serif, blue circle) as favicon and navbar/footer icon across all 8 pages of mynewweb.de.

**Architecture:** Create two SVG logo files (dark + light), generate favicon files from them, add CSS for sizing, then update all HTML pages with favicon `<head>` tags and the logo icon in navbar and footer.

**Tech Stack:** SVG, HTML5, CSS3 — no build tools required. Favicon .ico generated manually via realfavicongenerator.net.

## Task 3: Generate favicon.ico and apple-touch-icon.png (manual step)

**Files:**
- Create: `favicon.ico`
- Create: `apple-touch-icon.png`

These files cannot be generated by code — they require a manual step.

- [ ] **Step 1: Generate favicon.ico**

1. Open `images/logo-mnw.svg` in Chrome/Safari
2. Right-click → "Save image as" → save as `logo-mnw-32.png` (or take screenshot)
3. Go to **realfavicongenerator.net**
4. Upload the PNG
5. Download the generated package
6. Copy `favicon.ico` to the project root

- [ ] **Step 2: Generate apple-touch-icon.png**

1. Open `images/logo-mnw.svg` in browser
2. Take a 180x180 screenshot or export at 180x180 using any image tool
3. Add a `#0f172a` background (iOS needs non-transparent icons)
4. Save as `apple-touch-icon.png` in the project root

> **Note:** If you have Inkscape installed, you can run:
> `inkscape images/logo-mnw.svg --export-png=apple-touch-icon-temp.png --export-width=180 --export-height=180`
> Then add the dark background using any image editor.

- [ ] **Step 3: Verify files exist**

```bash
ls favicon.ico apple-touch-icon.png
```

Expected: both files listed.

## Task 5: Update index.html

**Files:**
- Modify: `index.html`

- [ ] **Step 1: Add favicon links to `<head>`**

Inside `<head>`, after the existing `<link rel="stylesheet">` tags, add:

```html
<link rel="icon" type="image/svg+xml" href="favicon.svg">
<link rel="icon" type="image/x-icon" href="favicon.ico">
<link rel="apple-touch-icon" href="apple-touch-icon.png">
```

- [ ] **Step 2: Add logo icon to navbar**

Find:
```html
<a href="index.html" class="navbar__logo">My<span>new</span>web</a>
```

Replace with:
```html
<a href="index.html" class="navbar__logo">
  <img src="images/logo-mnw.svg" alt="MNW" class="navbar__logo-icon">My<span>new</span>web</a>
```

- [ ] **Step 3: Add logo icon to footer**

Find:
```html
<div class="footer__logo">My<span>new</span>web</div>
```

Replace with:
```html
<div class="footer__logo">
  <img src="images/logo-mnw.svg" alt="MNW" class="footer__logo-icon">My<span>new</span>web</div>
```

- [ ] **Step 4: Verify in browser**

Open `index.html`. Expected:
- Browser tab shows MNW circle favicon
- Navbar shows small MNW circle icon left of "Mynewweb" text
- Footer shows same icon

- [ ] **Step 5: Commit**

```bash
git add images/logo-mnw.svg images/logo-mnw-light.svg favicon.svg favicon.ico apple-touch-icon.png css/style.css index.html
git commit -m "add MNW logo, favicon, and update index.html navbar/footer"
```
